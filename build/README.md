# Artificer Indicator

## Description

Simple mod to add an indicator similar to the keepsake indicator for when you have the Artificer active in your run. 

Maybe could be bundleted with other quality of life mods in the future.


## Installation

- Install from Thunderstore mod using any mod manager (e.g., r2modman)
